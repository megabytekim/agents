# Reading Club Experience Plugin

독서 경험을 풍부하게 만드는 두 가지 에이전트를 제공합니다.

## Agents

### 1. book-reading-partner
**책과 함께하는 독서 파트너**

PDF/EPUB 파일을 함께 읽고, 토론하고, 깊이 있게 이해하는 독서 파트너 에이전트입니다.

#### 주요 기능
- 📚 PDF/EPUB 파일 읽기 및 분석
- 💬 책 내용에 대한 깊이 있는 대화
- 🧠 대화 메모리 관리 (진도, 인사이트, 토론 주제)
- 🎯 맞춤형 독서 지원 (분석적/감성적/실용적)
- 📖 구체적인 텍스트 인용 및 참조

#### 사용 방법

1. **책 파일 준비**
   ```bash
   # PDF 또는 EPUB 파일을 input 디렉토리에 넣기
   cp your-book.pdf plugins/reading-club-experience/staging/input/
   ```

2. **에이전트 시작**
   ```bash
   # book-reading-partner 에이전트와 대화 시작
   agent book-reading-partner
   ```

3. **함께 읽기**
   - 에이전트가 자동으로 input 디렉토리의 파일을 감지합니다
   - 자유롭게 질문하고 토론하세요
   - 진도와 대화 내용은 `staging/memory/` 디렉토리에 자동 저장됩니다

#### 디렉토리 구조
```
plugins/reading-club-experience/
├── agents/
│   ├── book-reading-partner.md      # 독서 파트너 에이전트
│   └── book-experience-curator.md   # 체험 큐레이터 에이전트
├── staging/
│   ├── input/                       # 📚 PDF/EPUB 파일 넣는 곳
│   └── memory/                      # 🧠 대화 메모리 저장 장소
└── results/                         # 생성된 결과물
```

#### 대화 예시
```
당신: 안녕! 오늘 카프카의 변신을 읽으려고 해

에이전트:
안녕하세요! "변신"이시군요. 정말 강렬한 작품이죠.
처음 읽으시나요? 어떤 방식으로 함께 읽으면 좋을까요?
1. 차근차근 함께 읽으며 토론하기
2. 특정 부분에 대해 질문하기
3. 전체적인 테마와 해석 탐구하기

당신: 그레고르가 왜 가족을 떠나지 않았을까?

에이전트:
정말 좋은 질문이에요. 그레고르의 선택은 여러 층위로 해석할 수 있습니다.

첫째, 경제적 책임감입니다. 소설 초반에 그레고르는 부모님의 빚을 갚기 위해 일하고 있다고 나오죠...

[깊이 있는 토론 계속]
```

#### 메모리 기능
- 독서 진도 자동 추적
- 주요 토론 주제 기록
- 사용자의 인사이트와 해석 저장
- 다음 세션에서 맥락 복원

---

### 2. book-experience-curator
**책 기반 체험 큐레이터**

책의 주제와 테마를 분석하고, 한국에서 할 수 있는 관련 체험 활동을 추천하는 에이전트입니다.

#### 주요 기능
- 책의 핵심 주제와 테마 분석
- 관련 문화/예술 체험 추천
- 장소 기반 탐방 제안
- 독서모임 맞춤 활동 기획

#### 사용 방법
```bash
agent book-experience-curator

# 예: "채식주의자" 관련 체험 추천해줘
# 예: "82년생 김지영" 독서모임에서 할 만한 활동 알려줘
```

## 워크플로우 예시

### 깊이 있는 독서 여정
1. **book-reading-partner**로 책을 함께 읽고 토론
2. 핵심 주제와 테마 발견
3. **book-experience-curator**로 관련 체험 활동 추천받기
4. 독서모임에서 체험 공유 및 추가 토론

### 독서모임 준비
1. 다음 모임 책 선정
2. **book-reading-partner**로 미리 읽고 준비
3. **book-experience-curator**로 모임 활동 기획
4. 풍부한 독서 경험 완성

## 팁

### PDF/EPUB 파일 찾기
- 전자책 구매 플랫폼 (교보문고, 예스24, 리디북스 등)
- 공공 도메인 책 (구텐베르크 프로젝트 등)
- 도서관 전자책 대여 서비스

### 효과적인 대화 방법
- 구체적인 질문하기 (예: "3장에서 주인공이 왜 그런 선택을 했을까?")
- 자신의 해석 먼저 말하고 에이전트의 의견 듣기
- 이전 대화 내용 참조하기 (메모리가 기억합니다!)
- 책의 다른 부분과 연결해서 생각하기

### 메모리 활용
- 메모리 파일은 자동으로 생성 및 업데이트됩니다
- `staging/memory/` 디렉토리에서 직접 확인/수정 가능
- 여러 책을 동시에 읽어도 각각 독립적인 메모리 유지

## 지원 파일 형식
- ✅ PDF (.pdf)
- ✅ EPUB (.epub)
- 📝 추후 추가 예정: TXT, MOBI, AZW 등
