# Travel Curator Plugin

당신만의 여행 큐레이터 - 완벽한 여행을 설계하세요 🌍✈️

## 개요

Travel Curator는 여행 계획부터 구체적인 예약까지, 여행의 모든 단계를 도와주는 종합 여행 에이전트입니다.

## Agents

### 1. destination-explorer
**도시 탐험가 - 여행지 종합 가이드**

도시의 모든 것을 알려주는 현지 가이드 같은 에이전트입니다.

#### 주요 기능
- 🏛️ **도시 개요**: 역사, 문화, 특징
- 🎯 **꼭 해야 할 경험**: 명소, 액티비티, 숨은 보석
- 🍽️ **식당 추천**: 현지 맛집부터 고급 다이닝까지
- 🏨 **숙소 지역 가이드**: 지역별 특징 및 추천
- 💰 **예산 가이드**: 저예산부터 럭셔리까지
- 🚇 **교통 정보**: 공항에서 시내, 시내 이동
- ⚠️ **안전 및 팁**: 주의사항, 문화 예절

#### 사용 방법
```bash
agent destination-explorer

# 예시 질문
"이집트 여행 계획 중이야"
"카이로에 대해 알려줘"
"룩소르는 뭐가 유명해?"
```

---

### 2. experience-scout
**경험 스카우트 - 최적의 체험/숙소 찾기**

특정 투어, 액티비티, 숙소를 깊이 있게 검색하고 최고의 옵션을 찾아주는 에이전트입니다.

#### 주요 기능
- 🔍 **다중 플랫폼 검색**: GetYourGuide, Viator, Klook, Booking.com 등
- 💰 **가격 비교**: 여러 예약 사이트 가격 비교
- ⭐ **리뷰 분석**: 실제 여행자 리뷰 요약
- 📊 **비교 테이블**: 옵션별 장단점 한눈에
- 🎯 **추천도 점수**: 가격/품질/리뷰 종합 평가
- 💡 **프로 팁**: 현지인만 아는 팁
- 🔗 **예약 링크**: 최저가 예약처

#### 사용 방법
```bash
agent experience-scout

# 예시 질문
"룩소르 벌룬투어 찾아줘"
"카이로 피라미드 투어 비교해줘"
"아스완 5성급 호텔 추천해줘"
```

## 워크플로우 예시

### 여행 계획 전체 흐름

**Step 1: 도시 탐색**
```bash
agent destination-explorer
> "이집트 카이로 여행 가이드"
```
→ 카이로의 모든 것을 담은 종합 가이드 생성
→ `results/destinations/egypt-cairo-2025-12-08.md` 저장

**Step 2: 구체적 체험 찾기**
```bash
agent experience-scout
> "카이로에서 피라미드 투어 찾아줘"
```
→ 피라미드 투어 옵션 비교 (가격, 리뷰, 예약처)
→ `results/experiences/cairo-pyramid-tour-2025-12-08.md` 저장

**Step 3: 숙소 검색**
```bash
agent experience-scout
> "카이로 자말렉 지역 호텔 추천"
```
→ 호텔 옵션 비교 및 예약 링크
→ `results/experiences/cairo-zamalek-hotel-2025-12-08.md` 저장

## 특징

### 🌐 실시간 정보
- WebSearch로 최신 여행 정보 수집
- Playwright로 예약 사이트 크롤링
- 2024-2025년 최신 가격 및 리뷰 반영

### 💰 가격 투명성
- 여러 플랫폼 가격 비교
- 직접 예약 vs 플랫폼 예약 비교
- 숨은 비용 없이 투명하게

### 📊 데이터 기반 추천
- 실제 여행자 리뷰 분석
- 평점, 가격, 포함사항 종합 평가
- 추천도 점수 시스템

### 🗺️ 현지화된 정보
- 현지 문화 및 예절
- 안전 주의사항
- 현지인 추천 팁

### 📝 저장 및 공유
- 모든 결과를 마크다운으로 저장
- 여행 계획서로 활용 가능
- 친구들과 공유하기 쉬움

## 사용 팁

### destination-explorer 활용

**구체적으로 질문하기**
```bash
✅ "카이로 3일 일정 추천해줘"
✅ "룩소르에서 가족 여행하기 좋은 곳"
✅ "아스완 저예산 여행 팁"

❌ "이집트"  (너무 광범위)
❌ "여행"     (구체적이지 않음)
```

**결과 활용하기**
- 생성된 가이드를 여행 계획서로 사용
- 식당/명소 리스트를 구글 지도에 저장
- 예산 가이드로 여행 경비 계획

### experience-scout 활용

**검색 팁**
```bash
✅ "룩소르 벌룬투어 + 안전한 회사"
✅ "카이로 피라미드 프라이빗 투어"
✅ "아스완 나일강 크루즈 3일"

❌ "투어"     (너무 광범위)
❌ "호텔"     (도시/지역 명시 필요)
```

**비교 기준 설정**
- 예산 범위 말해주기
- 여행 스타일 (럭셔리/중간/백패커)
- 우선순위 (가격/품질/편의성)

## 지원 범위

### 여행지
- ✅ 전 세계 모든 도시
- ✅ 인기 관광지부터 숨은 명소까지
- ✅ 도시, 섬, 리조트 지역

### 체험 종류
- ✅ 투어 (도보, 버스, 프라이빗)
- ✅ 액티비티 (다이빙, 스카이다이빙, 벌룬)
- ✅ 문화 체험 (요리, 공예, 전통)
- ✅ 교통 (공항 픽업, 기차, 렌터카)

### 숙소 종류
- ✅ 호텔 (5성부터 게스트하우스)
- ✅ 리조트 (올인클루시브, 비치)
- ✅ 에어비앤비 (전체, 개인실)
- ✅ 특별 숙소 (사막 캠프, 보트)

### 예약 플랫폼
- GetYourGuide
- Viator
- Klook
- Booking.com
- Hotels.com
- Airbnb
- Expedia
- TripAdvisor
- 직접 예약 사이트

## 기술 스택

- **WebSearch**: 여행 정보, 리뷰, 가격 검색
- **Playwright**: 예약 사이트 크롤링 (WebSearch 실패 시)
- **Context7**: 여행 가이드북 정보 (가능한 경우)
- **Memory**: 사용자 여행 선호도 학습 (선택적)
